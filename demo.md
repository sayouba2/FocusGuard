# Short Demo

This repository includes a running Flutter prototype for Android.

Walkthrough:

1. Launch the app on Android.
2. Pick a session length like `25:00`.
3. Tap `Start`.
4. The timer counts down and the session status updates locally.
5. Tap `Pause` or `Reset` to control the session.

This is intentionally simple. The goal is to show the starting point of the project, not to pretend blocking is already complete.
